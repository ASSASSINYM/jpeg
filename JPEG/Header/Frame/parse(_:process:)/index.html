Static Method

parse(_:process:)

Parses a frame(_:) segment into a frame header.

Declaration

static func parse(_ data:[UInt8], process:JPEG.Process) throws -> Self

Parameters

data

The segment data to parse.

process

The coding process used by the image.

Return value

The parsed frame header.

Overview

If the given data does not parse to a valid frame header, this function will throw a ParsingError. This function invokes validate(process:precision:size:components:), so any errors it can throw can also be thrown by this function.
