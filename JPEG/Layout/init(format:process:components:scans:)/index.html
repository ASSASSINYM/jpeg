Initializer

init(format:process:components:scans:)

Creates an image layout given image parameters and a scan decomposition.

Declaration

init(format:Format, process:JPEG.Process, components:[JPEG.Component.Key: (factor:(x:Int, y:Int), qi:JPEG.Table.Quantization.Key)], scans:[JPEG.Header.Scan])

Parameters

format

The color format of the image.

process

The coding process used by the image.

components

The sampling factors and quantization table key for each component in the image.

scans

The scan progression of the image. All referenced image components must be recognized components.

Overview

If the image coding process is a sequential process, the given scan headers should be constructed using the sequential(...:) constructor. If the coding process is progressive, the scan headers should be constructed with the progressive(...:bits:), progressive(...:bit:), progressive(_:band:bits:), or progressive(_:band:bit:) constructors.

This initializer will validate the scan progression and attempt to generate a sequence of JPEG table definitions to implement the quantization table relationships specified by the components parameter. It will suffer a precondition failure if the scan progression is invalid, or if it is impossible to implement the specified table relationships with the number of table selectors available for the given coding process.
