Instance Property

definitions

The sequence of scan and table definitions in the image file.

Declaration

var definitions:[(quanta:[JPEG.Table.Quantization.Key], scans:[JPEG.Scan])] { get }

Overview

The definitions in this property are given as alternating runs of quantization tables and image scans. (Image layouts do not specify huffman table definitions, as the library encodes them on a per-scan basis.)
