Generic Structure

JPEG.Data.Rectangular

A rectangular image.

Declaration

struct Rectangular<Format> where Format:JPEG.Format

Overview

A rectangular image resamples all planes at the same sampling level, giving a rectangular array of interleaved samples.

It can be unpacked to various color targets to get a pixel color array.
