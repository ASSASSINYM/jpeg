Static Method

decompress(path:cosite:)

Decompresses a rectangular image from the given file path.

Declaration

static func decompress(path:String, cosite cosited:Bool) throws -> Self?

Parameters

path

A file path.

cosited

The upsampling method to use. Setting this parameter to true co-sites the samples; setting it to false centers them instead.

The default value is false.

Return value

The decompressed image, or nil if the file could not be opened at the given file path.

Overview

This function is a convenience function which calls Planar.decompress(path:) to obtain a planar image, and then calls interleaved(cosite:) on the output to return a rectangular image.

This function is only available on MacOS and Linux platforms.
