Generic Static Method

pack(size:layout:metadata:pixels:)

Packs the given row-major pixels into rectangular image data and creates a rectangular image with the given image parameters and layout.

Declaration

static func pack<Color>(size:(x:Int, y:Int), layout:JPEG.Layout<Format>, metadata:[JPEG.Metadata], pixels:[UInt16]) -> Self where Color:JPEG.Color, Color.Format == Format

Parameters

size

The size of the image, in pixels. Both dimensions must be positive.

layout

The layout of the image.

metadata

The metadata records in the image.

pixels

An array of pixels, in row major order, and without padding. The array must have exactly size.x × size.y pixels.

Return value

A rectangular image.

Overview

Passing an invalid size, or a pixel array of the wrong count will result in a precondition failure.

This function is the inverse of unpack(as:).
