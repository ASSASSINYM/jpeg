Instance Method

index(forKey:)

Returns the index of the plane storing the color channel represented by the given component key, or nil if the component key is a non-recognized component.

Declaration

func index(forKey ci:JPEG.Component.Key) -> Int?

Parameters

ci

The component key.

Return value

The integer index of the plane, or nil. If not nil, this index can be used with the [_:] subscript.
