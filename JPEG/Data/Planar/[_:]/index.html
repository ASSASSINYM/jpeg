Subscript

[_:]

Accesses the plane at the given index.

Implements requirement in RandomAccessCollection

Declaration

subscript(p:Int) -> Plane { get set }

Parameters

p

The index of the plane to access. This index must be within the index bounds of this RandomAccessCollection.

Return value

The plane.

Overview

The getter and setter of this subscript yield the plane using read and modify.
