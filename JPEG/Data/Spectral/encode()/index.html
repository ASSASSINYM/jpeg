Instance Method

encode()

Creates a frame header for this image.

Declaration

func encode() -> JPEG.Header.Frame

Return value

The encoded frame header.

Overview

The encoded frame header contains only the recognized components in this image. It encodes the image height eagerly (as opposed to lazily, with a HeightRedefinition header).
