Generic Instance Method

with(ci:_:)

Calls the given closure on the plane and associated quantization table for the given component key.

Declaration

mutating func with<R>(ci:JPEG.Component.Key, _ body:(inout Plane, JPEG.Table.Quantization) throws -> R) rethrows -> R

Parameters

ci

The component key of the plane to access. This component must be a recognized component, or this function will suffer a precondition failure.

body

The closure to apply to the plane and associated quantization table. Its return value is the return value of the surrounding function.

Return value

The return value of the given closure.

Overview

The closure passed to this method can mutate the plane in this image specified by the component key. The associated quantization table is still immutable, because editing it would also affect all other planes referencing that table.
