Instance Method

set(height:)

Sets the height of this image, in pixels.

Declaration

mutating func set(height y:Int)

Parameters

y

The new height of this image, in pixels. This height is measured from the top of the image. Passing a negative value will result in a precondition failure.

Overview

Existing data in this image will either be preserved, or cropped. Any additional data units created by this function will have all coefficients initialized to zero.
