init(arrayLiteral...:)

Creates a bitstream from the given array literal.

Implements requirement in ExpressibleByArrayLiteral

Declaration

init(arrayLiteral:UInt8...)

Parameters

arrayLiteral

The raw bytes making up the bitstream. The more significant bits in each byte come first in the bitstream. If the bitstream does not correspond to a whole number of bytes, the least significant bits in the last byte should be padded with 1-bits.

Overview

This type stores the bitstream in 16-bit atoms. If the array literal does not contain an even number of bytes, the last atom is padded with 1-bits.
