Static Method

parse(_:)

Parses an EXIF segment into a metadata record.

Declaration

static func parse(_ data:[UInt8]) throws -> Self

Parameters

data

The segment data to parse.

Return value

The parsed metadata record.

Overview

If the given data does not parse to a valid metadata record, this function will throw a ParsingError.

This constructor will attempt to index the root, EXIF (tag 34665), and GPS (tag 34853) field groups, if they are present and well-linked in the segment data. This parser ignores embedded thumbnails.
