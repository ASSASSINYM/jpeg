Subscript

[_:as:]

Loads the uint8 value at the given address.

Declaration

subscript(offset:Int, as _:UInt8.Type) -> UInt8 { get }

Parameters

offset

The address of the value to access. This pointer must be within the index range of storage.

_

This parameter must be set to UInt8.self.

Return value

The uint8 value.
