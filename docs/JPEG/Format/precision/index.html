Instance Property

precision

The bit-depth of each component in this color format.

Required.

Declaration

var precision:Int { get }

Overview

The baseline coding process can only be used with color formats with a precision of 8. The extended(coding:differential:) and progressive(coding:differential:) coding processes can only be used with color formats with a precision of 8 or 12.
