Structure

JPEG.Scan

A type modeling one scan of a JPEG image.

Declaration

struct Scan

Overview

Depending on the coding process used by the image, a scan may encode a select frequency band, range of bits, and subset of color components.

Topics

Instance properties

let band:Range<Int>

The frequency band encoded by this image scan.

let bits:Range<Int>

The bit range encoded by this image scan.

let components:[(c:Int, component:Component)]

The descriptors for the components encoded by this scan, in the order in which they are interleaved within the scan.
